# Csharp-Learning-Projects

This repository contains a collection of small projects and exercises focused on building practical skills in C# and related technologies.

## Purpose

The goal of this repository is to reinforce fundamental programming concepts through hands-on practice, with an emphasis on clean code, organization, and real-world application.

## Repository Structure

### LINQ
Practical examples and exercises using LINQ for data manipulation and querying.

### MySQL
Projects involving database interaction, including queries, basic modeling, and CRUD operations.

### OOP (Object-Oriented Programming)
Implementations of core concepts such as encapsulation, inheritance, polymorphism, and abstraction.

## Technologies

- C#
- .NET
- MySQL
