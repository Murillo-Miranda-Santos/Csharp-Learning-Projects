# BankConsoleApp

BankConsoleApp is a simple C# console application that simulates basic banking operations.

This project was created as a learning exercise to practice Object-Oriented Programming (OOP) concepts in .NET.

## Features

- Check account balance
- Withdraw money
- Deposit money
- Simple terminal menu system
- Input validation

## Technologies

- C#
- .NET Console Application
- Object-Oriented Programming

## How to Run

Clone the repository:

git clone https://github.com/Murillo_Miranda_Santos/BankConsoleApp.git

Run the application:

dotnet run

This project demonstrates:

- Class separation
- Encapsulation
- Menu control logic
- Input validation
- Basic console interaction
